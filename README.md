TCP Socket Chat Application
A simple Java chat app that allows two users to communicate in real-time using TCP sockets. 
The app has a GUI built with Swing, where users can send and receive messages simultaneously.

Features

-Real-time chat between client and server
-Reliable, ordered message delivery via TCP
-Full-duplex communication
-Simple Swing-based GUI

Files

ServerUI.java — Server program
ClientUI.java — Client program

How to Run
1-Compile:javac ServerUI.java ClientUI.java
2-Run the server:java ServerUI
3-Run the client: java ClientUI

Start chatting!
